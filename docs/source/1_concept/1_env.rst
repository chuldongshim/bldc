개발환경
========

개발전략
------------

BLDC 위치제어를 최종 타겟으로 정의하며 단계적으로 개발을 진행한다.

1. 1단계에서는 단순히 hall sensor를 사용하여 bldc 모터 정/역 회전을 구현하고,
2. 2단계에서는 sensorless를 사용하여 bldc 모터 정/역 회전을 구현한 다음
3. 3단계에서 위치제어를 구현한다.
4. 4단계에서 제어 성능을 높이기 위한 알고리즘을 추가로 구현한다.
